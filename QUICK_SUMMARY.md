## 💻 أمثلة الاستخدام

### التحقق من الدور:
```typescript
const { isAdmin } = useAuth();
if (!isAdmin()) return <div>ممنوع</div>;
```

### التحقق من الصلاحية:
```typescript
const hasAccess = usePermission(PERMISSIONS.MANAGE_PRODUCTS);
if (!hasAccess) return <div>ممنوع</div>;
```

### حماية المسار:
```typescript
<ProtectedRoute requiredRole="admin">
  <AdminPanel />
</ProtectedRoute>
```
