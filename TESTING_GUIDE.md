## 🔐 اختبارات الأمان الأساسية

### اختبار 1: محاولة الوصول إلى `/admin` بدون تسجيل دخول

**الخطوات:**
1. اذهب إلى `http://localhost:3000/admin`
2. لاحظ ماذا يحدث

**النتيجة المتوقعة:**
- ✅ تُعاد للرئيسية
- ✅ لا تظهر لوحة التحكم
- ✅ لا ترى أي عناصر إدارية

**الحالة:**
```
[ ] ✅ نجح
[ ] ❌ فشل
```

---

### اختبار 2: محاولة الوصول كعميل

**الخطوات:**
1. انقر على "دخول"
2. اختر "تسجيل جديد"
3. أنشئ حساب جديد
   - البريد: `customer@test.com`
   - كلمة المرور: `test1234`
   - الاسم: `عميل اختبار`
   - الدور: `عميل`
4. اضغط دخول كعميل
5. اذهب إلى `/admin` في رابط المتصفح

**النتيجة المتوقعة:**
- ✅ صفحة "وصول مرفوض"
- ✅ رسالة واضحة
- ✅ زر العودة للرئيسية

**الحالة:**
```
[ ] ✅ نجح
[ ] ❌ فشل
```

---

### اختبار 3: دخول المسؤول

**الخطوات:**
1. انقر على "دخول"
2. اختر "دخول"
3. أدخل البيانات:
   - البريد: `admin@micro-tech.com`
   - كلمة المرور: `admin123`
4. اضغط دخول

**النتيجة المتوقعة:**
- ✅ تسجيل دخول ناجح
- ✅ عرض الاسم في الرأس
- ✅ ظهور زر لوحة التحكم
- ✅ ظهور ملف المستخدم

**الحالة:**
```
[ ] ✅ نجح
[ ] ❌ فشل
```

## 🛡️ اختبارات الحماية

### اختبار 4: إخفاء الأزرار الإدارية

**الخطوات:**
1. ادخل كعميل
2. انظر إلى رأس الصفحة

**النتيجة المتوقعة:**
- ✅ لا يوجد زر "دخول الإدارة"
- ✅ لا يوجد لوحة تحكم
- ✅ الأزرار الأساسية فقط

**الخطوات 2: ادخل كمسؤول**
1. سجل خروج
2. ادخل كمسؤول
3. انظر إلى رأس الصفحة

**النتيجة المتوقعة:**
- ✅ يظهر زر "دخول الإدارة" أو "لوحة التحكم نشطة"
- ✅ يمكنك فتح لوحة التحكم

**الحالة:**
```
[ ] ✅ نجح
[ ] ❌ فشل
```

---

### اختبار 5: حماية لوحة التحكم

**الخطوات:**
1. ادخل كمسؤول
2. اضغط على زر "دخول الإدارة"
3. يجب أن تعرض لوحة التحكم

**النتيجة المتوقعة:**
- ✅ عرض لوحة التحكم
- ✅ جميع التابات مرئية
- ✅ يمكنك إدارة المنتجات والطلبات

**الخطوات 2: جرب كعميل**
1. سجل خروج
2. ادخل كعميل
3. حاول الوصول إلى زر الإدارة

**النتيجة المتوقعة:**
- ✅ الزر غير موجود أو معطّل

**الحالة:**
```
[ ] ✅ نجح
[ ] ❌ فشل
```

```json
{
  "id": "...",
  "email": "admin@micro-tech.com",
  "displayName": "...",
  "role": "admin",
  "createdAt": "..."
}
```
